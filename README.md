# BombVoyage

BombVoyage는 Unity를 사용하여 개발된 그리드 기반의 전략 퍼즐 게임입니다. 플레이어는 보조 폭탄(Auxiliary Bomb)을 설치하여 적을 물리치고 스테이지를 클리어해야 합니다.

## 🎮 게임 소개

보드 위를 돌아다니는 적들을 폭탄을 이용해 제거하세요. 폭탄이 터지면 적들은 밀려나거나 제거됩니다. 보드의 경계는 서로 연결되어 있어 한쪽 끝으로 이동하면 반대쪽에서 나타나는 랩핑(Wrap) 시스템을 차용했습니다.

### 주요 기능
- **그리드 기반 이동**: 적과 폭탄은 격자(Grid) 위에서 상호작용합니다.
- **월드 랩핑 (Wrap Around)**: 화면의 끝과 끝이 연결되어 있어, 전략적인 이동과 폭발이 가능합니다.
- **스테이지 시스템**: JSON 데이터를 통해 정의된 다양한 스테이지를 플레이할 수 있습니다.
- **다양한 아이템**: 보조 폭탄 등을 활용하여 전략적 플레이가 가능합니다.

## 🛠 개발 환경

- **Unity Editor**: 6000.2.10f1
- **Language**: C#

## 📂 프로젝트 구조

- `Assets/Monoscript`: 주요 게임 로직 스크립트가 포함되어 있습니다.
    - `GameManager.cs`: 게임의 승패 및 전체적인 상태를 관리합니다.
    - `StageRoot.cs`: 스테이지 초기화 및 매니저 의존성을 주입합니다.
    - `Enemy.cs`, `EnemyManager.cs`: 적의 생성, 이동, 피격 로직을 처리합니다.
    - `BombManager.cs`: 폭탄 설치 및 폭발 로직을 관리합니다.
    - `BoardManager.cs`: 보드 그리드 및 좌표 변환을 담당합니다.
- `Assets/Resources/Json/Stage`: 스테이지 레벨 디자인 데이터가 저장됩니다.

## 🕹️ 플레이 방법

1. **폭탄 설치**: 마우스를 클릭하여 비어있는 그리드에 폭탄을 설치합니다.
2. **연쇄 작용**: 폭탄이 폭발하면 주변 적들에게 넉백(Knockback) 효과를 주거나 제거합니다.
3. **승리 조건**: 모든 적을 제거하면 스테이지를 클리어합니다.
4. **패배 조건**: 가진 폭탄을 모두 소진하고, 설치된 폭탄도 남아있지 않은 상태에서 적이 살아남으면 패배합니다.

## 🚀 설치 및 실행

1. 이 저장소를 클론합니다.
2. Unity Hub에서 **Add**를 클릭하고 프로젝트 폴더를 선택합니다.
3. **Unity 6000.2.10f1** 버전으로 프로젝트를 엽니다.
4. `Assets/Scenes` 폴더 안의 씬을 열고 플레이 버튼을 누릅니다.
